# 🎨 Online Banking Frontend

The frontend for the Online Banking System, built with React 19, TypeScript, and Vite. It provides a modern, responsive, and high-performance user interface for managing bank accounts and transactions.

## ✨ Highlights

- **Modern Dashboard**: A clean and intuitive layout for viewing balances and history.
- **Responsive Design**: Fully optimized for mobile and desktop using Tailwind CSS.
- **Data Flow**: Type-safe API integration using TypeScript across all services.
- **Fast Development**: Powered by Vite and Hot Module Replacement (HMR).

## 🛠️ Tech Stack

- **React 19**: Utilizing the latest React features and optimizations.
- **Tailwind CSS**: For utility-first styling and premium aesthetics.
- **Lucide React**: A beautiful icon set for fintech applications.
- **React Router 7**: For seamless client-side navigation.
- **React Hot Toast**: For smooth notification feedback.

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## 📁 Project Structure

- `src/components`: UI components and layouts.
- `src/pages`: Main application views (Dashboard, Login, Transfers, etc.).
- `src/api.ts`: API service layer for backend communication.
- `src/utils.ts`: Helper functions and formatting utilities.
- `src/hooks`: Custom React hooks for shared logic.
